Στόχος Διπλωματικής Εργασίας

Η δημιουργία μεθοδολογίας αυτόματης ανίχνευσης της γλώσσας κειμένου από εικόνα.

Συνοπτική Περιγραφή Διπλωματικής Εργασίας

Το πρόβλημα αυτόματης ανίχνευσης της γλώσσας ενός κειμένου γίνεται πιο πολύπλοκο όταν δεν διαθέτουμε το κείμενο αλλά εικόνα του, σκαναρισμένη ή φωτογραφία. Ωστόσο είναι πολύ σημαντικό θέμα όταν θέλουμε να στείλουμε την εικόνα στο ανάλογο σύστημα αναγνώρισης χαρακτήρων ή όταν θα θέλαμε να κατηγοριοποιήσουμε τα έγγραφα αυτόματα σε μια βιβλιοθήκη. Σκοπός της συγκεκριμένης διπλωματικής είναι η μελέτη του συγκεκριμένου χώρου και η δημιουργία μεθοδολογίας που θα εκτελεί αυτόματα την ανίχνευση της γλώσσας του εγγράφου για συγκεκριμένες γλώσσες.

Στόχος Διπλωματικής Εργασίας

Στόχος είναι η υλοποίηση ενός αλγορίθμου εντοπισμού κειμένου σε φωτογραφίες.

Συνοπτική Περιγραφή Διπλωματικής Εργασίας

Η πληροφορία κειμένου σε εικόνες είναι πολύ χρήσιμη για υψηλή ανάλυση εικόνας, όπως ανάκτηση εικόνας, robot navigation και robotic vision. Για την αναγνώριση κειμένου σε φυσικό τοπίο, θα πρέπει κατ 'αρχάς να γίνει εντοπισμός κείμενου, το οποίο είναι πολύ δύσκολο λόγω σύνθετου υπόβαθρου, διαφορετικές συνθήκες φωτισμού και άγνωστες εμφανίσεις κειμένου. Ο φοιτητής καλείται να μελετήσει την υπάρχουσα βιβλιογραφία και να επιλέξει αλγόριθμο εντοπισμού κειμένου σε εικόνα, που θα υλοποιήσει.

ΠΡΟΣΑΡΜΟΓΗ COMICS ΓΙΑ TABLET H ΚΙΝΗΤΟ
Επιβλέπων Καβαλλιεράτου Εργίνα
Αριθμός Φοιτητών Μέχρι τρεις
Στόχος Διπλωματικής Εργασίας

Στόχος είναι η υλοποίηση ενός συστήματος μετατροπής του comic.

Συνοπτική Περιγραφή Διπλωματικής Εργασίας

Τα Comics είναι πολύ δημοφιλή μεταξύ των ανθρώπων από όλο τον κόσμο και από διαφορετικές ηλικίες. Με την ανάπτυξη των κινητών συσκευών και δικτύων επικοινωνίας, όλο και περισσότεροι άνθρωποι επιθυμούν να διαβάσουν τα σαρωμένα αντίγραφα των κόμικς στις κινητές συσκευές τους, όπως κινητά τηλέφωνα ή tablet PCs. Ωστόσο, λόγω των περιορισμών απεικόνισης αυτών των συσκευών, το περιεχόμενο μιας ενιαίας σελίδας κόμικ είναι μεγαλύτερο από ότι μία κινητή συσκευή μπορεί να χωρέσει, και ως εκ τούτου είναι πολύ δύσκολο για τους αναγνώστες να διακρίνουν τους χαρακτήρες κειμένου στην οθόνη της κινητής συσκευής. Ο φοιτητής καλείται να μελετήσει την υπάρχουσα βιβλιογραφία και να επιλέξει σύστημα προσαρμογής κομικ, που θα υλοποιήσει.

ΑΝΑΣΥΓΚΡΌΤΗΣΗ ΣΧΙΣΜΕΝΩΝ ΕΓΓΡΑΦΩΝ

Αριθμός Φοιτητών Μέχρι τρεις
Στόχος Διπλωματικής Εργασίας

Στόχος είναι η υλοποίηση αλγόριθμου ανασυγκρότησης σχισμένων εγγράφων.

Συνοπτική Περιγραφή Διπλωματικής Εργασίας

Η ανασυγκρότηση σκισμένων έγγραφων έχει άμεση εφαρμογή σε forensic τομείς. Οι ειδικοί στο χώρο θα μπορούσαν να λάβουν τη βοήθεια της αυτοματοποίησης για να γίνει η δουλειά τους. Η ανοικοδόμηση σκισμένων εγγράφων είναι στενά συνδεδεμένη με το παραδοσιακή πάζλ από χαρτόνι. Η κύρια διαφορά μεταξύ ανασυγκρότησης σκισμένων έγγραφων και πάζλ είναι η παρουσία των παράτυπων, στο σχήμα, κομματιών εγγράφου και το περιεχόμενο. Ο φοιτητής καλείται να μελετήσει την υπάρχουσα βιβλιογραφία και να επιλέξει αλγόριθμο ανασυγκρότησης σχισμένων εγγράφων, που θα υλοποιήσει.
